## Edge DSD

Edge Distributed Security Detection

#### 功能

- 网站安全检测
- 分布式扫描

#### 依赖

* Scrapy >= 1.0.0
* redis-py >= 2.10.0
* redis server >= 2.8.0

#### 组件

* 调度器
* 重复任务过滤
* 解析与分析
* 基础请求与下载


#### 安装

  $ git clone https://github.com/edge-security/edge_dsd.git
  $ cd edge_dsd
  $ python setup.py install
